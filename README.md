# Portfólio — João Leal · Product Builder

Site pessoal construído com vibe coding: Next.js 16 (App Router, 100% estático), Tailwind CSS v4, deploy na Vercel.

## Editar conteúdo

Todo o texto vive em `content/` (`site.ts`, `home.ts`, `crm.ts`) — tipado, sem CMS. Componentes não contêm copy.

## Rodar

```bash
npm install
npm run dev
```

## Prints do CRM

Salve os prints em `public/crm/` com os nomes de `docs/prints-roteiro.md` — a galeria e o card da home detectam os arquivos no build.
